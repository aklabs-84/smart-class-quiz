# TaskList

## 개요
현재 코드 기준으로 완료/부분/대기 항목을 정리했습니다.

| 로비 화면 | 참가자 목록 표시/애니메이션 | ✅ 완료 |
| 퀴즈 진행 | 4지선다, 타이머 표시 | ✅ 완료 |
| 결과 화면 | 정답 표시 및 선택 통계 | ✅ 완료 |
| 순위 화면 | 랭킹 표시, 최근 점수 반영 | ✅ 완료 |
| 최종 결과 | 최종 점수/리셋 UI | ✅ 완료 |
| 게임 제어 | 시작/결과/순위/다음 문제 전환 | ✅ 완료 |
