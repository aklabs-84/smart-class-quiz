# PRD (Product Requirements Document)
# Smart Class Quiz - 실시간 수업 퀴즈 웹앱

## 1. 제품 개요

### 1.1 제품명
Smart Class Quiz (Kahoot 스타일 실시간 수업 퀴즈)

### 1.2 제품 설명
교실에서 선생님이 실시간으로 퀴즈를 진행하고, 학생들이 동시에 참여해 점수와 순위를 확인하는 웹앱입니다. 선생님/학생 모드를 분리해 진행 흐름을 단순화했고, Google Sheets 연동 API가 설정된 경우 실시간 데이터 동기화를 사용합니다. API가 없는 경우에는 목(Mock) 데이터로 데모 진행이 가능합니다.

### 1.3 제품 목표
- 수업 참여도 향상과 즉시 피드백 제공
- 선생님 중심의 간단한 진행 UX
- 설치 없이 브라우저에서 바로 참여
- Google Sheets 기반의 손쉬운 데이터 운영

---

## 2. 사용자 정의

### 2.1 선생님 (교사/진행자)
- 비밀번호로 선생님 모드에 प्रवेश
- 퀴즈 시작/결과/순위/다음 문제 전환
- 문제별 타이머 설정 및 BGM 제어
- 참여자 현황 및 응답 집계 확인

### 2.2 학생 (참가자)
- 이름 입력만으로 참여
- 문제 풀이 및 결과 확인
- 현재 점수/순위 확인

## 4. 화면 구성 (현재 구현 기준)

### 4.1 시작/모드 선택
- 선생님/학생 버튼으로 모드 선택

### 4.2 선생님 로그인
- 비밀번호 입력 후 대시보드 진입

### 4.3 선생님 대시보드
- 로비/퀴즈/결과/순위/최종 결과 화면 제어
- 응답 현황 및 참여자 수 표시

### 4.4 학생 화면
- 이름 입력 후 로비 대기
- 퀴즈 진행 중 답변 선택
- 결과/순위/최종 결과 확인

---

## 5. 데이터 구조 (요약)

- 참가자: 이름, 참여 시간, 점수, 답변 기록
- 문제: 질문, 보기 4개, 정답, 제한 시간
- 응답: 참가자, 문제, 선택 답안, 정답 여부, 점수, 응답 시간
- 게임 상태: 현재 단계, 현재 문제 번호, 제한 시간, 세션 ID

---

## 6. 비기능 요구사항

### 6.1 성능/동기화
- 실시간 동기화는 폴링 기반
- 참가자/상태/응답은 초 단위 주기로 갱신

### 6.2 호환성
- 데스크톱/모바일 브라우저 지원
- 반응형 레이아웃

### 6.3 사용성
- 한국어 UI
- 설명 없이도 진행 가능한 단순 흐름

---

## 7. 제약사항 및 범위

- 별도 백엔드 로그인 시스템 없음
- PIN 코드 참여 방식은 현재 미구현
- Google Sheets 연동은 API 설정 여부에 따라 동작
