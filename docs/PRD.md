# PRD (Product Requirements Document)
# Smart Class Quiz - 실시간 수업 퀴즈 웹앱

## 1. 제품 개요

### 1.1 제품명
Smart Class Quiz (Kahoot 스타일 실시간 수업 퀴즈)

### 1.2 제품 설명
교실에서 선생님이 진행하는 실시간 인터랙티브 퀴즈 플랫폼입니다. 학생들이 동시에 참여하여 퀴즈를 풀고, 실시간으로 순위를 확인할 수 있습니다. Google Sheets와 연동하여 문제 관리, 참가자 관리, 결과 저장을 쉽게 할 수 있습니다.

### 1.3 목표
- 수업 참여도 향상
- 실시간 피드백을 통한 학습 효과 증대
- 무료이고 쉬운 설정으로 누구나 사용 가능
- Google Sheets 연동으로 간편한 데이터 관리

---

## 2. 사용자 정의

### 2.1 선생님 (교사/진행자)
- 퀴즈 게임을 생성하고 진행하는 주체
- 학생 참여 현황을 모니터링
- 게임 시작/종료 권한 보유
- 문제별 제한 시간 설정 가능

### 2.2 학생 (참가자)
- 이름을 입력하여 게임에 참여
- 주어진 시간 내에 문제를 풀고 답안 선택
- 실시간으로 자신의 순위 확인

## 6. 비기능 요구사항

### 6.1 성능
- 페이지 로드 시간: 3초 이내
- 실시간 데이터 동기화: 2초 이내
- 동시 접속자: 최소 50명 지원

### 6.2 호환성
- 데스크톱 브라우저 (Chrome, Firefox, Safari, Edge)
- 모바일 브라우저 (iOS Safari, Android Chrome)
- 반응형 디자인 필수

### 6.3 보안
- 선생님 계정 비밀번호 보호
- Google Sheets API 키 보안 관리

### 6.4 사용성
- 직관적인 UI로 설명 없이 사용 가능
- 한국어 인터페이스
- 시각적 피드백 제공

## 7. 제약사항

- 무료 서비스 사용 (Google Sheets API, 무료 호스팅)

- 실시간 동기화는 폴링 방식으로 구현
